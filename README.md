# famous-game
既存の有名なゲームを作ってみる

## 概要 (Overview)
このリポジトリでは、様々なプログラミング言語で有名なゲームを実装します。  
各実装は学習目的で作成され、異なる言語での実装方法を比較できるようになっています。

## ゲーム一覧 (Games)

### 三目並べ (Tic-Tac-Toe)
- **Python実装**: `python/tic-tac-toe/`
- **JavaScript実装**: `javascript/tic-tac-toe/`

## 使い方 (How to Use)

各ゲームのディレクトリには、それぞれの言語での実装と実行方法が記載されたREADMEがあります。

### Python版を実行
```bash
cd python/tic-tac-toe
python tic_tac_toe.py
```

### JavaScript版を実行
```bash
cd javascript/tic-tac-toe
node tic_tac_toe.js
```

## 開発環境 (Development Environment)
- OS: macOS
- IDE: VSCode
