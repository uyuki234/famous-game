# 三目並べ (Tic-Tac-Toe) - Python版

## 概要
コマンドラインで動作するシンプルな三目並べゲームです。

## 特徴
- 2人対戦モード
- わかりやすいコマンドライン表示
- 入力検証とエラーハンドリング
- 日本語インターフェース

## 必要な環境
- Python 3.6以上

## 実行方法

```bash
python tic_tac_toe.py
```

または

```bash
python3 tic_tac_toe.py
```

## ゲームの遊び方

1. ゲームを起動すると、位置番号が表示されます
2. プレイヤーXとプレイヤーOが交互に手番を行います
3. 1から9の数字を入力して、マスを選択します
4. 縦、横、斜めのいずれかで3つ並べると勝利です
5. すべてのマスが埋まっても勝者が決まらない場合は引き分けです

## ボード配置

```
 1 | 2 | 3 
---|---|---
 4 | 5 | 6 
---|---|---
 7 | 8 | 9 
```

## ゲームルール
- プレイヤーXが先攻です
- 既に置かれているマスには置けません
- 無効な入力の場合は再入力を求められます

## コード構造
- `TicTacToe` クラス: ゲームロジックを管理
- `display_board()`: 現在のボード状態を表示
- `check_winner()`: 勝者判定
- `play()`: ゲームメインループ
