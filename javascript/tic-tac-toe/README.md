# 三目並べ (Tic-Tac-Toe) - JavaScript版

## 概要
Node.jsで動作するコマンドラインの三目並べゲームです。

## 特徴
- 2人対戦モード
- わかりやすいコマンドライン表示
- 入力検証とエラーハンドリング
- 日本語インターフェース
- 非同期処理を使用した入力管理

## 必要な環境
- Node.js 12.0以上

## 実行方法

```bash
node tic_tac_toe.js
```

または実行権限を付与して:

```bash
chmod +x tic_tac_toe.js
./tic_tac_toe.js
```

## ゲームの遊び方

1. ゲームを起動すると、位置番号が表示されます
2. プレイヤーXとプレイヤーOが交互に手番を行います
3. 1から9の数字を入力して、マスを選択します
4. 縦、横、斜めのいずれかで3つ並べると勝利です
5. すべてのマスが埋まっても勝者が決まらない場合は引き分けです

## ボード配置

```
 1 | 2 | 3 
---|---|---
 4 | 5 | 6 
---|---|---
 7 | 8 | 9 
```

## ゲームルール
- プレイヤーXが先攻です
- 既に置かれているマスには置けません
- 無効な入力の場合は再入力を求められます

## コード構造
- `TicTacToe` クラス: ゲームロジックを管理
- `displayBoard()`: 現在のボード状態を表示
- `checkWinner()`: 勝者判定
- `play()`: ゲームメインループ（async/await使用）
- readline モジュールを使用したユーザー入力処理

## 技術的な特徴
- ES6+ の機能を使用（クラス、async/await、アロー関数等）
- Promise ベースの非同期入力処理
- モジュールとしてエクスポート可能
